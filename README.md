🚀 Personal Portfolio Website
This repository contains the source code for my professional portfolio website. It is designed to showcase my journey as an AIML student, my technical skills, and the various projects I've built, including AI-driven logistics solutions and interactive simulations.

Live Demo: https://bhushan806.github.io/Portfolio-/

🎨 Design & Features
Modern UI: A clean, dark-themed interface built with a focus on readability and professional aesthetics.

Responsive Design: Fully optimized for all screen sizes, from mobile devices to large desktop monitors.

Interactive Elements: Smooth navigation and interactive project cards.

Project Showcases: Dedicated sections for major projects like TruckNet India and AgroIndia Game.

🛠️ Built With
HTML5: For structured and semantic web content.

CSS3: Custom styling featuring a dark-mode palette and responsive layouts.

JavaScript: For dynamic interactions and smooth navigation.

📂 Project Structure
Plaintext

├── style.css         # Custom styling and layout definitions
├── script.js        # Interaction logic (if applicable)
└── assets/           # Images, icons, and media files
